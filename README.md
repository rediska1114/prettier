# prettier

## install:

```bash
npm i -D prettier
```

## script

```ts
 "prettier": "prettier --write src/**/*.*"
```

## add `.prettierrc`

```json
 "arrowParens": "avoid",
 "bracketSpacing": true,
 "htmlWhitespaceSensitivity": "strict",
 "jsxBracketSameLine": false,
 "jsxSingleQuote": true,
 "printWidth": 100,
 "quoteProps": "as-needed",
 "semi": false,
 "singleQuote": true,
 "tabWidth": 2,
 "trailingComma": "es5",
 "useTabs": true
```
